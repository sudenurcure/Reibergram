# Calculation
Function: QIgX = (a/b)*(Q^2Alb+ b^2)^(1/2)-c 
Upper limit IgG:
a/b = 0.93
b^2 = 6
c = 1.7

Lower limit IgG:
a/b = 0.33
b^2 = 2
c = 0.3
